# 00_PROJECT_RULES

## 1. 프로젝트 정의

HWASEO MARKET은 전통시장의 점포와 상품을 함께 보여주며,
온라인 판매와 오프라인 방문을 동시에 유도하는 시장 공식 웹몰이다.

## 2. 핵심 목표

- 점포 홍보와 상품 판매를 동시에 수행하는 홈 구조 구축
- 쿠폰, 이벤트, 특가 중심의 행동 유도형 배너 운영
- 화서시장 지도 기반 점포 탐색 경험 제공
- 점포 정보와 상품 정보를 연결하여 구매/방문 전환 유도
- 운영자가 점포/상품/이벤트를 쉽게 수정 가능한 구조 확보

## 3. 홈 핵심 구조

- 대형 배너(쿠폰/이벤트/혜택)
- 시장 지도 기반 점포 탐색
- 점포 홍보 + 상품 목록 혼합 섹션
- 공지/시장 소식
- 신뢰 보강 요소
- 푸터

## 4. 반드시 포함할 것

- 점포와 상품이 함께 보이는 홈 구조
- 지도 마커 기반 점포 정보 인터랙션
- 온라인 구매 CTA
- 오프라인 방문/쿠폰 CTA
- 운영 가능한 점포/상품/배너 데이터 구조

## 5. 금지사항

- 일반 식품몰형 상품 나열 구조 회귀 금지
- 지도 없는 베스트 상품 메인 구조 금지
- 점포 정보 없는 상품 노출 남발 금지
- 점포를 독립 브랜드 플랫폼처럼 과도 확장 금지
- final/new/copy/temp/back-up 파일명 금지
