# 02_HOME_SECTION_SPEC

## HeroEventBanner

- 목적: 쿠폰/이벤트/혜택을 가장 먼저 노출
- 허용 콘텐츠:
  - 신규 쿠폰
  - 시장 행사
  - 방문 혜택
  - 온라인 주문 특가
  - 시즌 기획전
- 필수 요소:
  - 배너 이미지
  - 이벤트 태그
  - 제목
  - CTA 버튼
  - 연결 링크

## MarketMapSection

- 목적: 지도 기반 점포 탐색
- 필수 요소:
  - 화서시장 지도 이미지 또는 SVG
  - 점포 마커
  - 마커 hover/touch/click 반응
  - 점포명
  - 위치
  - 한줄 소개
  - 점포 연결 CTA
  - 관련 상품 또는 대표 상품 정보
- 주의:
  - 단순 이미지 배치 금지
  - 지도는 탐색 도구여야 함

## StoreAndProductSection

- 목적: 점포 홍보와 상품 판매를 동시에 수행
- 필수 요소:
  - 점포 카드 또는 점포 강조 블록
  - 대표 상품 목록
  - 점포 한줄 소개
  - 구매 CTA
  - 점포 보기 CTA
- 주의:
  - 상품만 따로 분리된 일반 쇼핑몰 섹션처럼 보이면 안 됨

## NoticeSection

- 목적: 시장 소식 전달
- 필수 요소:
  - 공지 제목
  - 날짜
  - 중요 표시

## TrustSection

- 목적: 배송, 신뢰, 결제, 친환경, 방문 혜택 등 보강
